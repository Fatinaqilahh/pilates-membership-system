# Pilates Membership System

This is a PHP & MySQL-based Pilates Membership System developed using XAMPP.

## Features
- Public Pilates website
- Member sign up & login
- Class booking system
- Membership plans
- Admin dashboard

## Requirements
- XAMPP
- PHP 7+
- MySQL
- Browser

## Setup Instructions
1. Clone the repository
2. Copy project folder to `htdocs`
3. Import database using provided SQL file
4. Start Apache & MySQL in XAMPP
5. Access via `http://localhost/pilates_system`
